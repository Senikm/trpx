Detailed Description

template<typename Iter>
class jpa::Bit_range< Iter >

Bit_range<T> defines a view of a range of Bit<T> in a container<T>, providing an interface with the STL. It also allows packing and unpacking integral types into and out of a Bit_range<T>.

IterThe iterator type.
