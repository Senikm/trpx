Detailed Description

template<typename T>
class jpa::Grey_tif_image< T >

API to an image managed by a regularized Grey_tif object with a compile time defined pixel type.

This class provides methods for manipulating and accessing pixel data of an image in a Grey_tif object. It is an STL compatible range view, and inherits and provides all the features of std::span. It has no public constructors and can only be created by a Grey_tif object to allow access to its TIFF image data. If the In case of raw TIFF data (stored in a Grey_tif<std::byte> object), the size() member function returns zero if the template parameter does not correspond to the type encoded in the raw TIFF data.
