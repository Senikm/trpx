Detailed Description

template<typename Iter>
class jpa::Bit< Iter >

The Bit class represents a reference to a bit in an object of type *T.

It allows retrieving and manipulating the referenced bit.

Template Parameters
IterA random access iterator or pointer referring to an integral type.
