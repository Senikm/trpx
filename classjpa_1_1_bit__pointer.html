Detailed Description

template<typename Iter>
class jpa::Bit_pointer< Iter >

Bit_pointer<T> is an STL pointer to a Bit<T> and supports all pointer operators. T is an iterator of an integral type. If a pointer operator causes a Bit_pointer to point beyond the size of *T, it moves to the next T-type object as defined by the pointer arithmetic of container<T>.

Template Parameters
IterType of the iterator to which this Bit_pointer points.
